Sobre mim

Olá! Sou Pedro Marcolino, estudante de Engenharia de Computação e apaixonado por tecnologia. Tenho interesse em desenvolvimento de software e gosto de transformar ideias em soluções reais, combinando lógica, design e criatividade.

Já desenvolvi projetos em linguagens como C e Assembly, além de aplicações modernas com React, React Native, Next.js, Node.js e MySQL. Também gosto de explorar o lado da eletrônica digital, criando circuitos e simulando sistemas lógicos.

Meu objetivo é continuar evoluindo como programador e engenheiro, criando soluções que unam eficiência, usabilidade e inovação.

Projetos

Criei um aplicativo de tarefas simples e funcional usando apenas HTML, CSS e JavaScript puro. O projeto permite adicionar, remover e marcar tarefas como concluídas, armazenando as informações no navegador através do localStorage. Esse app foi essencial para consolidar minha lógica de programação, manipulação do DOM e domínio das bases do JavaScript moderno sem o uso de frameworks.

Desenvolvi o App Guardian com React Native usando ExpoRouter, um aplicativo voltado à segurança e monitoramento pessoal. O objetivo do projeto foi criar uma solução prática e intuitiva para auxiliar o usuário em situações de emergência, com interface leve e responsiva. Nesse projeto, aprofundei meu conhecimento em desenvolvimento mobile multiplataforma, componentes nativos e integração com recursos do dispositivo.

Desenvolvi uma aplicação inspirada no Spotify utilizando React, com foco em recriar a interface moderna e interativa da plataforma. O projeto inclui componentes reutilizáveis, navegação fluida entre playlists e integração com dados dinâmicos simulados. Esse app foi uma ótima oportunidade para aprimorar meus conhecimentos em componentização, hooks e gerenciamento de estado, além de aplicar boas práticas de UI/UX no front-end.
